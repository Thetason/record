## 🔧 테스트 방법

### 로컬 테스트
```bash
# IP 주소 확인
ifconfig | grep "inet " | grep -v 127.0.0.1

# 개발 서버 실행
npm run dev

# 모바일에서 접속
http://[YOUR_IP]:3000
```

### Chrome DevTools
1. F12 → Toggle device toolbar
2. Responsive 모드 선택
3. 네트워크 스로틀링 (Slow 3G)
4. 터치 시뮬레이션 활성화

### 실제 디바이스 테스트
- iOS: Safari, Chrome
- Android: Chrome, Samsung Internet
- 가로/세로 모드 전환
- 다크 모드 지원
